📦 Registry

Ready-to-use modules for your GUIDE.md

The GUIDE.md registry provides pre-configured modules that you can extend in your own GUIDE.md files. Install with guidemd add <module> or use extends: "<module>" in your frontmatter.

How to Use

Method 1: Using the CLI

$ guidemd add <module-name>

This installs the module into your GUIDE.md's modules field and merges its frontmatter.

Method 2: Using extends

extends: "typescript-strict" # or multiple extends: - "typescript-strict" - "testing-jest"

Add to your frontmatter. Values in your file override the module's values.
