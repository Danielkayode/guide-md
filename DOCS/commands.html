CLI Commands

Complete reference for the @guidemd/linter CLI

Core Commands

guidemd lint [file] Core
Validate a GUIDE.md file against the spec (handles inheritance).
$ guidemd lint GUIDE.md

Flags

--json Output results as JSON for CI/tooling integration
--fix Automatically fix fixable issues
--sync Detect and sync drift between frontmatter and project files
--stats Output Context Density Score
--skip-secret-scan Skip scanning for secrets (not recommended)
✔ GUIDE.md is valid This file is ready to be consumed by AI agents and MCP servers.
guidemd init
Scaffold a new GUIDE.md in the current directory with smart stack detection.
$ guidemd init --force

Flags

--force Overwrite an existing GUIDE.md
guidemd sync [file] Core
Detect and sync drift between frontmatter and actual project files.
$ guidemd sync GUIDE.md --readme README.md

Flags

-r, --readme <path> Path to README.md for bi-directional sync

Export & Import

guidemd export [file] Export
Export GUIDE.md to other AI context formats.
$ guidemd export GUIDE.md --target all

Flags

-t, --target <type> Target format: claude, cursor, windsurf, agents, copilot, aider, all
-o, --out <dir> Output directory (default: .)
-m, --manifest Generate MCP manifest.json
guidemd import <file> Import
Reverse-parse an AI context file (CLAUDE.md, .cursorrules, etc.) into a GUIDE.md.
$ guidemd import CLAUDE.md --out GUIDE.md

Flags

-o, --out <path> Output path for the generated GUIDE.md
--dry-run Print to stdout instead of writing file

Analysis Commands

guidemd info [file] Analysis
Display a high-level health report of the project's AI-readiness.
$ guidemd info GUIDE.md --json

Flags

--json Output results as JSON
guidemd optimize [file] Analysis
Analyze GUIDE.md for token efficiency and structural improvements.
$ guidemd optimize GUIDE.md --json

Flags

--json Output results as JSON
guidemd doctor [file] Analysis
Deep static analysis to find logic conflicts and architectural drift.
$ guidemd doctor GUIDE.md
guidemd verify [file] Analysis
Verify GUIDE.md provides enough context for AI cold start.
$ guidemd verify GUIDE.md --json

Flags

--json Output results as JSON

Comparison

guidemd diff [file] [compare] Comparison
Compare two GUIDE.md files and show what changed.
$ guidemd diff GUIDE.md GUIDE.md.backup
$ guidemd diff GUIDE.md --git
$ guidemd diff GUIDE.md --git --breaking

Flags

--git Diff against git HEAD instead of a file
--breaking Only show changes that affect AI agent behavior
--json Output as JSON instead of formatted text

Utility Commands

guidemd badge Utility
Generate a Markdown badge for your README (Dynamic Grading).
$ guidemd badge --file GUIDE.md
guidemd ci Utility
Generate a GitHub Action workflow template.
$ guidemd ci --write

Flags

--write Create .github/workflows/guidemd.yml directly
guidemd watch [file] Utility
Watch mode: re-run lint on every save of GUIDE.md.
$ guidemd watch GUIDE.md

Registry Commands

guidemd add <module> Registry
Install a registry module into your GUIDE.md.
$ guidemd add typescript-strict
guidemd list Registry
List available registry modules.
$ guidemd list
